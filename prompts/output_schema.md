# Output Schema — 热点梗分析输出格式

## Markdown 输出格式（给人看的报告）

每个热点梗的分析输出必须包含以下结构：

```
**热梗标签**：#标签1、#标签2

**热点视频**：
- https://www.douyin.com/video/xxxxx
- https://www.douyin.com/video/xxxxx

**梗分析**：
（1-3 句话。说清楚这个梗是什么、梗的核心玩法/句式/动作是什么、为什么适合做广告片头引流。）

**可参考的内容产出方向**：
①方向名称：
  示例：具体的视频脚本一句话描述
  - 具体画面/操作描述；
②方向名称：
  - 具体画面/操作描述；
```

## 字段规范

| 字段 | 要求 | 示例 |
|------|------|------|
| 热梗标签 | 1-3 个抖音话题标签，用顿号分隔 | #bro懂我的穿搭、#bro懂我的枪法 |
| 热点视频 | 1-5 个抖音视频 URL，每行一个 | https://www.douyin.com/video/7511321099213884682 |
| 梗分析 | 50-150 字，不超过 3 句话 | "bro懂我的xx"梗完美契合CF手游的竞技性… |
| 内容产出方向 | 1-3 个方向，每个方向含具体示例 | ①游戏实况：示例："bro懂我的瞬狙"… |

## 约束

- **每条分析总长度**：200-500 字（不含 URL）
- **视频链接**：必须是完整的 douyin.com/video/ 格式 URL，不给无效链接
- **如果没有采集到视频**：在"热点视频"处写明"暂无采集到的视频链接，建议在抖音搜索以下关键词：xxx"
- **内容产出方向**：必须和游戏/广告素材相关，不给泛内容建议
- **禁用词汇**：不得出现"结合自身定位""打造差异化""多角度切入""值得关注""不容错过"等空话
