# 🔥 抖音热点梗分析工具

> 面向星广联投素材团队的抖音热点采集 + AI 分析工具。
> 一键获取当下最火的梗，AI 自动给出游戏广告素材的创作方向。

---

## 🎯 怎么用？（给运营同学看这里）

### 你不需要装任何东西，只需要跟你的 AI Agent 对话。

**第一次使用，复制下面这段话发给你的 Agent：**

> 帮我安装并配置「抖音热点梗分析」工具。
> GitHub 地址：https://github.com/caicai688/douyin-hotspot
> 请把它安装到 `~/.workbuddy/skills/douyin-hotspot`，安装好所有 Python 依赖，然后帮我做首次配置。

Agent 会自动帮你完成：
- ✅ 下载代码
- ✅ 安装 Python 依赖（requests、playwright、google-genai）
- ✅ 安装浏览器引擎
- ✅ 配置 API Token
- ✅ 引导你扫码登录热点宝（第一次需要你用抖音扫码）

### 示例对话

**你：** 帮我跑一份今天的抖音热梗分析报告，分析 8 个话题，用 AI 分析

**Agent 会自动：**
1. 采集抖音热点宝三大榜单（搜索榜、视频榜、话题榜）
2. 智能过滤掉不适合做素材的话题（时政、灾难等）
3. 合并同一事件的多个话题（不浪费分析名额）
4. 用 Gemini AI 逐个分析每个热梗
5. 输出报告（包含梗解读 + 创作方向 + 示例文案 + 镜头建议）

---

## 📄 报告里有什么？

每个话题的分析包含：

- **热梗标签**：提炼出的核心梗标签
- **梗分析**：这个梗是什么、怎么火的
- **可参考的创作方向**：2-3 个方向，每个方向有——
  - 具体的内容思路
  - 示例文案
  - 分镜/镜头描述

末尾有一张 **梗参考汇总表**，方便快速查阅所有梗。

## ⚙️ 需要提前准备的东西

| 项目 | 说明 | 怎么获取 |
|------|------|---------|
| **Gemini API Key** | AI 分析用 | 打开 https://aistudio.google.com → 登录 Google 账号 → Get API key → Create → 复制 Key |
| **抖音账号** | 热点宝扫码登录用 | 用你常用的抖音 APP 扫码即可 |

> Gemini 免费额度完全够用，不需要花钱。
> 如果不提供 Key，工具也能跑，只是分析质量会差一些（降级为规则分析）。

## 🔧 常见问题

**Q: 报告显示"规则分析"而不是 AI 分析？**
→ 让 Agent 帮你配置 Gemini API Key。

**Q: 话题榜数据很少或报错？**
→ 热点宝 Cookies 过期了，对 Agent 说："帮我重新登录热点宝"，然后扫码。

**Q: 分析速度慢？**
→ 正常，AI 分析每个梗需要 10-30 秒。8 个话题大概 5-8 分钟。

**Q: 想重新分析今天已分析过的话题？**
→ 对 Agent 说："重新跑一份热梗报告，不要跳过已分析的"。
